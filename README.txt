This is the Unidata Sruth data-distribution package from the Unidata Program
Center of the University Corporation for Atmospheric Research. This package
enables the rapid dissemination of arbitrary files over the Internet via a
publish/subscribe distribution model integrated with an asynchronous,
event-driven, peer-to-peer transport mechanism.

The official website for this package is
<http://www.unidata.ucar.edu/projects/sruth>.
