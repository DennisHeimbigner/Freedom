Unidata SRUTH Project

Perform the following steps to test the Unidata SRUTH project:

  1. Install Java 7 on your system. Java 7 is not yet a production release.
  2. Install the Unidata Data Distribution system. Click on one of the following and remember the installation directory:
    • Unix installation. Have your browser use the sh shell to open the installer.
    • Windows installation.

  3. Use the Unidata SRUTH client (SRUTHclient) to subscribe to NEXRAD Level II radar data from station KFTG (Denver). Have your browser open files of mime type application/x-sruth with the just-installed SRUTHclient utility, which is in the installation directory of the previous step.
